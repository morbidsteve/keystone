🔍 SAST

Semgrep

Pattern-based static analysis with OWASP Top 10, CWE Top 25, and 5 custom Keystone rules targeting SQL injection, hardcoded secrets, sensitive logging, pickle deserialization, and SSRF.
Fails: ERROR severity Gate: SAST check NIST SA-11 OWASP V1.14
🔑 SECRETS

TruffleHog

Scans entire repo history for leaked credentials, API keys, and tokens. Only verified secrets trigger failure to reduce false positives.
Fails: Any verified secret Gate: Secrets check CWE-798
🦠 MALWARE

ClamAV

Virus and malware scanning across the entire repository. Zero tolerance — any infection immediately fails the pipeline.
Fails: Any infection Gate: Malware check NIST SI-7
📜 LICENSE

Syft + License Policy

Generates SBOM (SPDX 2.3 + CycloneDX), then validates against allowed-licenses.yml. Blocks 8 copyleft licenses (GPL, AGPL, SSPL, EUPL). Warns on 7 weak copyleft.
Fails: Blocked license Gate: License check NIST CM-14 SLSA L2
🔬 SCA

Grype

CVE vulnerability scanning against SBOM for container images. Runs in CI (Phase 2) and nightly rescans. Unfixed CVEs are not ignored.
Fails: Critical CVEs Gate: Image scan NIST RA-5 CIS 4.1
🔬 CONTAINER

Trivy

Container image vulnerability scanner. Defense-in-depth alongside Grype. SARIF output uploaded to GitHub CodeQL Security tab.
Fails: CRITICAL + HIGH Gate: Image scan NIST RA-5 CIS 4.1
⚡ DAST

OWASP ZAP

Dynamic application security testing against ephemeral CI stack. API scan via OpenAPI spec + full-site crawl of the frontend.
Fails: High/Critical Gate: DAST check NIST RA-5 OWASP V2/V5
✍️ SIGNING

Cosign (Sigstore)

Keyless OIDC image signing and SBOM attestation. Runs on main branch and release tags only. Required for production deployment.
Fails: Signature failure Gate: Production release NIST SI-7 SLSA L2
🧪 TEST

pytest + Vitest

Backend: pytest with PostgreSQL 15 + PostGIS + Redis service containers. Frontend: Vitest for React/TS. Both require 80% minimum coverage.
Fails: <80% coverage Gate: Test checks NIST SA-11
📦 BUILD

Docker Buildx

Multi-stage container builds with OCI labels, build provenance, and GHA caching. Pushes to GitHub Container Registry (GHCR).
Fails: Build error Gate: Implicit SLSA L2
🏗️ HARDENING

Container Hardening

Validates DoD CHG v1.2: non-root (UID≥1000), read-only rootfs, no-new-privileges, drop ALL caps, healthcheck, no :latest, signed + SBOM.
Fails: Policy violation Gate: Image scan DoD CHG v1.2 CIS 4.6
🔄 PR GATE

Dependency Review

GitHub-native PR gate. Blocks merges introducing critical-severity dependencies or banned licenses. Posts summary comment on every PR.
Fails: Critical / banned Gate: PR merge NIST CM-14
🌙 MONITORING

Nightly Rescan

Daily 6AM UTC rescan of latest main images using Syft + Grype + Trivy. Auto-creates deduplicated GitHub Issues on new vulnerabilities. 90-day artifact retention.
Fails: Critical (default) Gate: Advisory NIST RA-5
🔍 LINT

Ruff + mypy

Python backend linting (Ruff) and type checking (mypy). Enforces formatting standards and catches type errors before build.
Fails: Any lint/type error Gate: Code Quality OWASP V1.14
